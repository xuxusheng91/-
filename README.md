 # 缘舍 · 静心祈福平台
 
 > 一念清净，福慧双修。以互联网之力，弘扬传统文化，让每个人都能随时随地随缘祈福。
 
 ## 项目概况
 
 | 项目 | 说明 |
 |------|------|
 | 项目名称 | 缘舍 (Yuan She) |
 | 类型 | 静心祈福平台 |
 | 技术栈 | Next.js (App Router) |
 | 部署地址 | https://deploy-yuanshe.vercel.app |
 | 当前状态 | 首页可用，子页面路由 404 |
 
 ## 项目结构
 
 yuanshe/
   README.md            项目说明
   src/                 源代码目录
   docs/analysis/       站点分析报告
   docs/design/         设计方案
   docs/roadmap/        开发计划
   assets/images/       图片资源
   assets/styles/       样式文件
   scripts/             工具脚本
   notes/               工作笔记
 
 ## 服务功能（计划）
 
 - 祈福 - 为家人、朋友或自己祈福，点燃心灯
 - 线上香 - 在云端供奉一炷清香
 - 黄历 - 查看每日宜忌、吉时
 - 灵签 - 虔诚求签，寻解心中疑惑
 - 八字分析 - 通过生辰八字分析命理运势
 - 占卜 - 六爻、梅花易数等传统占卜方式
 - 解梦 - 周公解梦，解读梦境中的启示
 - 手相 - 观手相而知运势
 - 起名 - 结合八字五行起名
 - 禅坐 - 跟随引导冥想禅坐
 
 ## 快速开始
 
 git clone <repo-url>
 cd yuanshe
 npm install
 npm run dev
 npm run build
 
 ## 部署
 
 当前通过 Vercel 部署：https://deploy-yuanshe.vercel.app
 需在 Vercel 项目中配置 vercel.json 的 Rewrites 规则以支持 SPA 客户端路由。
