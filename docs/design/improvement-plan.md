 # 美化与设计改进方案
 
 ## 现状评价
 
 首页已有不错的视觉基底：
 - 中国风配色（金色/红色/米白）
 - 清晰的层级结构
 - 响应式导航
 
 ## 改进方向
 
 ### 1. 视觉升级
 
 - 引入毛笔字字体（如 Noto Serif SC）
 - 增加微交互动效（hover 放大、涟漪效果）
 - 优化渐变和阴影层次
 - 增加云纹、莲花等传统文化装饰元素
 - 背景纹理（宣纸质感 SVG 图案）
 
 ### 2. 动画与过渡
 
 - 页面切换使用淡入淡出
 - 卡片入场使用交错滑入动画
 - 祈福灯、点香等核心交互增加粒子效果
 - 滚动触发渐入动画
 
 ### 3. 色彩方案
 
 主色调：
 - 朱红 #C03A2B
 - 金色 #C9A84C
 - 米白 #F5F0EB
 - 墨色 #2C2C2C
 - 檀木 #8B6914
 
 ### 4. 组件优化
 
 - 卡片增加微妙的纸质感（box-shadow + 边框）
 - 按钮统一 rounded-lg + hover 效果
 - 导航激活态高亮
 - 统一 Icon 风格
 - 加载状态 Skeleton 组件
 
 ### 5. 响应式细节
 
 - 大屏：两栏/三栏布局展示服务卡片
 - 平板：自适应网格
 - 手机：单栏 + 底部 Tab Bar
 
 ### 6. 禅意氛围
 
 - 可选背景音乐（古琴/梵音）
 - 极简禅意 loading 动画
 - 动态粒子背景（莲花瓣/香云）
